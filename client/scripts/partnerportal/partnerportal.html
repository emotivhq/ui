Adding the GiftStart It button is easy. Pick your platform:

  • Shopify

To add the button to your Shopify store, you can either add it site-wide (for all products), or to individual products.

• To add the button to all products, click "Online Store", then "Themes", then "Edit HTML/CSS" under the "...". Next, click "product.liquid" under your Templates, and add the following code on a new line after "{% include 'product' %}":

• or, to add the button to a single product, click the "<&rt;" symbol in the editing bar for the Description of your product, then paste the following code (and click "Save"):

                <gs-button id="gsbutton" class="gsbutton"></gs-button>
                <script>
                    window.giftStartButton = {
                        publicKey: '{{partner.api_key}}'
                    };
                </script>
                <script src="https://www.giftstarter.co/scripts/shopify/button.js"></script>
                <style>
                #gsbutton {
                    height: 36px;
                    border: 0;
                    background-color: white;
                }
                #gsbutton a img {
                    border-radius: 2px;
                }
                </style>

You can modify the styling if you like, but be sure to only adjust the height or the width - doing both will likely distort the button!

To add the button to a site, you should insert the following snippet next to your "Add to Cart" button:

                <gs-button id="gsbutton" class="gsbutton" style="display: none;"></gs-button>
                <script>
                    window.giftStartButton = {
                        productUrl: 'http://example.com/product/12345',
                        title: 'Example title',
                        price: 85.00,
                        imgUrl: 'http://example.com/images/12345.jpg',
                        publicKey: '{{partner.api_key}}'
                    };
                </script>
                <script src="https://www.giftstarter.co/scripts/button/general.js"></script>

You'll need to replace the example product info in the first script tag with your actualy product data. On most platforms this can be done in the template for your product details page.

You should style the button to match your existing buttons height and corner rounding, but a good starting point for the buttons CSS is:

                #gsbutton {
                    height: 36px;
                    border-radius: 4px;
                    background-color: white;
                }

Be sure to only adjust the height or the width - doing both will likely distort the button!
